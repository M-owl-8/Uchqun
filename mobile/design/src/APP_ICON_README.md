## 🔄 How to Use These Icons

### For iOS & Android:
1. Open SVG in design software (Figma, Adobe Illustrator, Sketch)
2. Export as PNG at these sizes:
   - **iOS:** 1024x1024, 512x512, 180x180, 120x120, 87x87, 80x80, 76x76, 60x60, 58x58, 40x40, 29x29, 20x20
   - **Android:** 512x512, 192x192, 144x144, 96x96, 72x72, 48x48, 36x36
3. Use a tool like [AppIcon.co](https://appicon.co) to auto-generate all sizes

### For Expo/React Native:
1. Save your chosen icon as `icon.png` (1024x1024)
2. Place in `/assets/` folder
3. Update `app.json`:
```json
{
  "expo": {
    "icon": "./assets/icon.png"
  }
}
```

### For Web:
1. Export as PNG at 512x512
2. Convert to favicon sizes using [RealFaviconGenerator](https://realfavicongenerator.net)

## 💡 Customization Notes

Want to adjust the icons? Here are some easy changes:

1. **Make child more abstract:** Remove facial features for broader representation
2. **Add accessibility symbols:** Include wheelchair, hearing aid, etc.
3. **Change background color:** Swap Warm Sand for pure white or gradient
4. **Adjust heart style:** Make it more rounded or geometric
5. **More/fewer sparkles:** Adjust quantity based on style preference
6. **Different pose:** Change arm positions for different emotions
