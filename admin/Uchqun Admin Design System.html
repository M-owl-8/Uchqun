Uchqun · School Steward Portal

A warm-professional design language for the school director's office.

The Admin portal is operated by school stewards in Uzbekistan — directors of special-education institutions who are part operator, part caretaker, part civic actor. This document defines the visual system that distinguishes Admin from the Government portal: terracotta over institutional green, cream over cool paper, papers-on-a-desk over flat inspector forms.

Inter · base 16px Terracotta brand Cream + warm-neutrals Subtle shadows on cards 8/10 px radii lucide @ 1.75 stroke

3 — Migration order

  1. ①
    Tokens first. Land the config above. Build green. No screens change yet.
  2. ②
    Shared components second. Update @shared/components/ Button, Card, Badge, Input, Modal, Toast, Tabs, PageHeader, ConfirmDialog, EmptyState, Table to consume the new tokens.
  3. ③
    Page-by-page third. Sidebar & Layout → Login → Dashboard → ReceptionManagement → (new) Documents queue → (new) AI Warnings → remaining pages.

  • •  All Uzbek copy on these reference screens is plausible production copy — feel free to lift verbatim into locales/uz/common.json.
