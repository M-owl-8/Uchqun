Visual design language · Reception portal

Warm operational.
A well-organized front desk.

Reception is the hospitality face of the school and its highest-volume data-entry surface. This reference defines the colours, surfaces, motifs and components used by front-desk staff at Uzbek special-education schools — warm without being childish, dense without being cold, culturally grounded without becoming decoration.

For implementation

This document is the source of truth for the Reception portal's visual language. Tokens live in reception/tailwind.config.js as a local extend on top of the shared base preset — they are not added to the shared preset because other portals keep their own brand scales.

reception/tailwind.config.js local extend 
// Reception portal — local theme extend. Inherits olive success.* from shared base.
theme: { extend: {
  colors: {
    paper:   '#FBFAF6',
    surface: '#FFFEFB',
    brand: { /* 50 → 900, primary = 600 = #C89030 */ },
    slate: { /* warm slate, 50 → 900 = #1F2528 */ },
    teak:    { DEFAULT: '#2A3B3D', hover, divider, muted, text },
    accent:  { 100, 300, 500: '#5A8A87', 700 }, // muted teal-sage
    warning: { /* pure amber #C58A1F — distinct from brand */ },
    error:   { /* #9A3E3A — consistent with Admin   */ },
    info:    { /* #4D6584 — consistent with Admin   */ },
    // success.* inherited from shared preset — do NOT redefine
  },
  boxShadow: { xs, sm, md, lg, xl /* see Tokens · Shadow */ },
  borderRadius: { sm:'6px', md:'8px', lg:'12px', xl:'16px' },
  fontFamily: { sans: ['Inter', ...], /* enable cv11, ss01, tnum */ },
}}

Patterns & motifs

Government and Admin avoid decorative pattern. Reception introduces it — atmospherically, never on data surfaces. If a reviewer can't immediately tell why a pattern is there (what job it does), remove it.

A · Uzbek-inspired rhombus tile 24×24 px sidebar · 60×60 px login

Abstracted from traditional ikat / suzani geometry — stripped to pure repeating rhombi, single-colour, 1-px stroke. Functions as identity texture, not visible decoration.

Pattern fades to 0 % before the sidebar nav items begin — implemented as a mask-image: linear-gradient(to bottom, #000, transparent) on the motif layer. Never appears on data surfaces.
